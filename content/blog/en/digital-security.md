# International Standard Enterprise Data Security

Information security is a top priority at WG Technology. We are committed to protecting your data according to the strictest standards.

## Security Standards
- **ISO/IEC 27001**: Information Security Management System.
- **SOC 2 Type 2**: Internal controls for security and availability.
- **GDPR**: Compliance with General Data Protection Regulation.

## Security Features in DocuWare
1. **Data Encryption**: AES-256 bit for data at rest and TLS for data in transit.
2. **Access Control**: Granular permissions down to individual documents and fields.
3. **Audit Trail**: Logs all user activities on the system for auditing purposes.
